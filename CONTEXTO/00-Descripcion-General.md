# Sistema Mini Bufete de Abogados

## Descripción General

El Sistema Mini Bufete de Abogados es una plataforma web que permite gestionar servicios legales de manera digital.

Los clientes pueden registrarse, solicitar consultas legales, agendar citas, subir documentos, realizar pagos y hacer seguimiento de sus casos.

Los abogados pueden administrar los casos asignados, revisar documentación, registrar observaciones, actualizar estados y generar reportes.

Los administradores son responsables de gestionar usuarios, abogados, clientes, roles, permisos y configuraciones del sistema.

## Objetivo

Digitalizar los procesos del bufete para mejorar la organización, reducir tiempos de atención y facilitar la comunicación entre clientes y abogados.

## Actores Principales

* Cliente
* Abogado
* Administrador

## Tecnologías Propuestas

* Frontend: Angular
* Backend: Spring Boot
* Base de Datos: PostgreSQL
* Arquitectura: Microservicios
* Diagramación: PlantUML y C4 Model
* Control de Versiones: GitHub
