# Nivel 4 - Código

En este nivel se describe una vista detallada de clases y relaciones del microservicio de casos.

Se recomienda usar el archivo `C4-Codigo-CaseService.puml` para representar las entidades, servicios y repositorios más relevantes del dominio de casos.

Esta vista ayuda a comprender el diseño de clases, los patrones de persistencia y cómo el servicio de casos encapsula la lógica principal del dominio.
