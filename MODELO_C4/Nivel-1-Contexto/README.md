# Nivel 1 - Contexto del Sistema

Este diagrama muestra el contexto global del sistema IURIS y sus interacciones con los actores principales y los sistemas externos.

Actores:
- Cliente
- Abogado
- Administrador
- Gerente

Componentes clave:
- Portal Web (Angular)
- Aplicación Móvil (Flutter)
- API Gateway (Spring Cloud Gateway)
- Cloudinary
- Servicio de Videollamadas (WebRTC / Jitsi)

El objetivo es representar cómo los usuarios acceden al sistema y cómo se integra con servicios externos para documentos y videollamadas.
